## Name That Color - Android Studio / IntelliJ plugin

When I get a new color to set in my app, I never know how to call it. I used [Name That Color](http://chir.ag/projects/ntc/) for years, and I wanted to have it integrated inside Android Studio, so I ported that JS library to Kotlin and built this plugin on it.

## Usage

Have the color you want in your clipboard (copy it from whatever) and use auto complete to automatically insert the color resource with its name. Magic!
