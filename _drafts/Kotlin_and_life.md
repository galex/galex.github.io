[Kotlin](http://kotlinlang.org/) is a new programming language for the JVM, Javascript, and in the (near) future, native (iOS, Windows). If you didn't try Kotlin yet, [there's a really good in-browser IDE](https://try.kotlinlang.org/) to try it online (Nothing to install).

I heard about it a long time ago but didn't really tried it until [Google announced Kotlin became a first class citizen language on Android](https://www.youtube.com/watch?time_continue=6&v=X1RVYt2QKQE). I was genuinely happy to hear this news for two reasons:

1. No more excuses to **not** learn it
2. Businesses will be much more inclined to let us, tech people, **use it in production**

So it was decided. I will learn the shit of this language! The question is how? First, just [try it out]((https://try.kotlinlang.org/)). Then read the [documentation reference](http://kotlinlang.org/docs/reference/). I would not call it an easy read but it gives an overview of what is possible to do with Kotlin.
